Communicate Data Findings for the dataset 'Ford GoBike System Data'¶

by Ekaterina Litvinova¶

Investigation Overview¶

In this analysis, the main interest is the data associated with the user, as well as the features associated with the movement of the users. I find it useful for business to have an information about the most popular stations, user routes, and the hours and days of the week when users are most active in rental.

Dataset Overview¶

The data for this dataset was downloaded from Udacity. The data set includes information about individual rides made in a bike-sharing system in February, 2019. The total amount of data items (rides) in the dataset after cleaning all incorrect data is 174952.

Count of rides by hour, weekday by user type¶

So now on the first graph we can see that regardless of the user type the most popular time is to pick up the bike is around 8 and 17 o'clock. Based on this we can conclude that majority of rides is to and from the office (or work place).
On the second graph there's also a similar distribution between two user types and the busiest day is Thursday, which is quite unexpected.

The relationship of the length of the journey with the age of the user by dividing the users into categories¶

Based on the graph we can conclude that trip duration for Subscrbers is almost equally distributed up to age of 63 years and then decreases. For Customers the diference is more noticable and the longest rides are by users if ages between 20 and 40. We can also see that Customers have more long trips than Subscribers, which we also observed on a box plot previously.
